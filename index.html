Team 7 YouTube Video Classification Project

Github Linkedin Demo

Team 7, We classified YouTube videos into educational and non-educational and then further categorized them into different categories such as business, coding ,IT and Software, etc. We used ML.NET/C#/Python/YouTube Data API to do this. You can download the application from the link below

After downloading the zipped folder, unzip it, and open the EducationalModel.exe file to use the educational/non-educational classifier and CategorizeModel.exe file for the sub-categorized classifier.
